Glosario
=========

.. glosary::

    API
        *Aplication Programing Interfaces* es una definición de un contrato para utilizar la funcionalidad proveida
        por un sistema

    HAR
        *Human Activity Recognition* son las siglas en inglés del proceso de reconocer actividades físicas utilizando
        sensores y dispositivos móviles
