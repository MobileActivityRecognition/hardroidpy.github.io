HARDROID
########

Human Activity Recognition For Android
======================================

Bienvenido a la documentacion de HARDroid!
------------------------------------------

Este proyecto es un servicio de reconocimiento de actividades humanas
para Android. Utiliza dos componentes de librería y un APK de servicio
para reutilizar el código.

Puedes ver el abance del proyecto en el repositorio `GitHub <https://github.com/hardroidpy>`_  y 
verificar la documentación `JavaDoc <javadoc/index.html>`_ para saber más detalles del
funcionamiento de la :term:`API`.
